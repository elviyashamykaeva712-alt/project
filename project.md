# Игра «Камень-ножницы-бумага»

## Описание
Консольная игра «Камень-ножницы-бумага» на Python с сохранением статистики.

## Правила
- Камень побеждает ножницы
- Ножницы побеждают бумагу
- Бумага побеждает камень

## Управление
- 1 - Камень
- 2 - Ножницы
- 3 - Бумага
- 0 - Выход из игры

## Запуск
```bash
python game.py
